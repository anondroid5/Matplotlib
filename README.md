# Matplotlib
pylabはmatplotlibのオブジェクト指向ライブラリに対する手続き的インターフェース

# Environment
python 2.7

download following file.

http://peak.telecommunity.com/dist/ez_setup.py

easy_install SET UP

```
python ez_setup.py
```

PIP INSTALL

```
easy_install pip
```

Important Libraries INSTALL

```
pip install ipython
```

```
easy_install readline
```

```
pip install numpy
```

```
pip install matplotlib
```
